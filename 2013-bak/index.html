第二波徵文活動開始囉！

鑑於許多朋友反應上一次文章徵稿時間太短而來不及準備，因此我們再次邀請大家一起來用文章參與 PHPConf ！

活動日期

即日起至 8/23 號接受投稿，我們將於 8/26 日通知您的文章是否入選。

徵求主題

如果您的文章內容符合以下主題，我們將優先收錄！

  • PHP 5.3, 5.4, 5.5 特色應用
  • PHP 程式架構與佈署
  • PHP 開發工具
  • PHP 開源應用軟體模組開發
  • 巨量資料處理

其他 PHP 相關主題也歡迎大家投稿。

投稿方式

請依照以下格式，將您的稿件寄到：cota@phpconf.tw 。

投稿格式

  • 文章主題 (五十字以內)
  • 作者姓名 (網路暱稱亦可)
  • 目前服務單位與職稱 (若不便透露的話，可略過)
  • 主題目標對象 (分為初學者，五年以下開發經驗的開發者，五年以上開發經驗的開發者)
  • 文章內容 (兩千字以內，若包含說明圖片時，圖檔格式為 300 dpi 之 PNG 檔)
  • 授權方式 (請參考下方文章著作權說明)

入選獎勵

一旦您的主題入選，我們將會贈送您大會 T-shirt 一件及 VIP 入場門票一張。

主題說明

今年 PHPConf 著重的主軸將是實戰經驗分享與 PHP 新特色應用，而且我們也希望 PHP 開發不僅僅只是 PHP 本身，而是能與其他技術相輔相成。因此我們選擇了以下幾個主題，提供各位在投稿時參考。

系統佈署

如何能夠讓 PHP 應用程式能快速佈署到各式平台上對 PHP 開發者而言是一件苦惱的事情！若是您有非常棒的 PHP 佈署技術，或是慘痛的 PHP 佈署甘苦談，都將是不錯的分享題目。

Extension 開發

加速 PHP 的方法有很多，但最直接的一種就是開發 PHP Extension 。而在開發 PHP Extension 的過程中也有很多面向需要注意，例如測試與除錯。如果您在這方面有豐富經驗，我們歡迎各位來分享您的甘苦談。

開發工具

近年來， PHP 已經有許多不輸其他語言的方便工具可以使用，它們可以為我們的開發節省許多時間；我們歡迎各種介紹與展示這些新工具的主題，當然自製的工具也沒問題。

特色應用

PHP 自 5.3 版之後，新增了許多有趣且實用的特色，像是命名空間、匿名函式等等；如果您已經在實務應用中體驗到這些新特色的便利或缺陷，我們都歡迎您來分享。

模組開發

目前世界上已經有很多 PHP 開源應用程式被廣泛地使用，許多企業也相繼以這些開源應用程式來製作自家網站。我們歡迎對開源 CMS 套件模組開發有實務經驗的高手，前來分享您的經驗。

巨量資料

當網站資料流越來越大時，如何有效率地讓頁面快速呈現這些龐大的資訊？加機器？改程式？您對處理巨量資料有獨到的做法嗎？您又是如何透過 PHP 達成這個目標呢？我們歡迎您來分享您的實戰經驗！

ABOUT 2013

一直以來，大多數 Web 製作者若是想要學習後端開發，立刻聯想到的語言就是 PHP ； PHP 讓很多人從此踏入 Web 開發的世界且樂此不疲，這是因為 PHP 易學、易用，而且參考資源相當豐富；也因此對許多初學者而言，它是友善且自由的。而從許多國際知名的大型網站 (例如 Yahoo! 、 Facebook ) 是建構在 PHP 之上來看，就足以瞭解 PHP 對於 Web 開發的優勢所在。

然而也因為 PHP 一開始是專注在 Web 開發上，而非朝向真正的語言發展，導致它為此背負了許多舊有的包袱；時至今日，還是讓很多開發者對它抱有很大的誤解。不過近年來， PHP 社群意識到了這些問題，進而提出許多改革方案，使得 PHP 漸漸脫胎換骨，重新出發！

今年， PHPConf 將要為大家帶來新面貌的 PHP ，主題將涵蓋資安、核心、工具、效能、 CMS 模組開發等等。我們將強調以實戰經驗為主，以實例帶領大家親身體驗 PHP 的威力。開發者可以從中學習到業界高手是如何撰寫並善用 PHP ；而企業更可以透過這場聚會，建立起與各方領域高手交流的管道。因此，今年的 PHPConf 是您絕對不能錯過的重要聚會！

Orange Tsai		 在網路資訊安全越顯重要的今天，您如何保護您的網站資料不被竊取呢？瞭解駭客的心理與技術，瞭解網站程式有何漏洞，都是現在 Web 開發必須熟知的課題。這次， PHPConf 邀請到了 Allen Own 與 Orange Tsai 兩位台灣資安界的高手，他們將透過精心設計的攻防實戰，告訴大家資安的重要性！
TOOLS c9s JavaScript 、 CSS 等這類前端檔案的最小化及相依性管理，常是 Web 開發者頭痛的問題之一。雖然可以借助 Framework 來處理，但也相對地要讓程式碼綁死在 Framework 上面。 PHPConf 邀請到台灣 PHP 界的第一把交椅 c9s ，他將為大家介紹他自己設計的工具： AssetToolkit ，來更有效率地管理前端檔案。
EXTENSION Rack Lin 效能問題在 PHP 上一直都是很值得探討的議題，而提升執行效能的其中一種方法，就是將關鍵部份以 C 來開發；可是 PHP Extension 的開發對一般開發者來說，門檻相當高；因此 PHPConf 邀請到了技術界人稱阿土伯的 Rack Lin ，他將介紹如何透過 Phalcon Framework 的核心，開發屬於自己的 PHP Extension ！
BIG DATA tsung 當網站規模逐漸擴大時，開發者就一定會面臨到大流量、大資料量的處理。而在台灣，這樣的環境與經驗通常只掌握在大型企業手中，一般開發者很少能接觸到。 PHPConf 這次邀請到 tsung ，他將透過開發 SiteTag 的經驗，來為大家說明如何處理巨量資料。
